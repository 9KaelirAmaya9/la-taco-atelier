# Kitchen Staff Access Guide

## Quick Access URL
**Kitchen Staff Login:** `your-domain.com/kitchen-login`

## How to Access the Kitchen Orders System

### For Kitchen Staff:
1. **Navigate to:** `/kitchen-login`
2. **Enter your credentials:**
   - Email address (provided by admin)
   - Password (provided by admin)
3. **Click "Sign In to Kitchen"**
4. You'll be automatically redirected to the kitchen orders page

### Features:
- ✅ **Secure Login** - Only kitchen staff and admins can access
- ✅ **Auto-redirect** - If already logged in, goes straight to kitchen
- ✅ **Role Verification** - Checks for kitchen or admin role before allowing access
- ✅ **Clear Error Messages** - Shows specific errors if login fails
- ✅ **Mobile-Friendly** - Works on tablets and phones

### Important Notes:
- Only users with "kitchen" or "admin" role can access
- Session persists - staff won't need to login repeatedly
- Contact admin if you don't have credentials

### Admin: How to Create Kitchen Staff Accounts
1. Go to `/admin` (admin login required)
2. Navigate to "Role Management" tab
3. Add "kitchen" role to user accounts
4. Share credentials with kitchen staff

### Troubleshooting:
- **"Access denied"** - User doesn't have kitchen role assigned
- **"Login failed"** - Check email and password are correct
- **Stuck loading** - Refresh the page and try again
